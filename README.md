## What's This All About?

Living in Bhilai and tired of waiting at bus stops without knowing when the next bus will arrive? Or maybe you're a student at BIT College trying to figure out the cheapest way to get home? This app has got your back!

I built this because navigating public transport in our region can be a real pain. Whether you're commuting to the Steel Plant, heading to Maitri Bagh for a weekend outing, or catching a train from Durg station, this app makes it super easy.

## For the Tech-Savvy Folks

This is built as a web application that works on any device - your phone, laptop, or tablet. Here's what's under the hood:

- **Frontend**: Pure HTML, CSS, and JavaScript (no fancy frameworks needed!)
- **Backend**: Node.js with Express (for the full-featured version)
- **Database**: PostgreSQL (stores all route and schedule data)
- **Maps**: Integrated with location services for real-time tracking

### Want to Run the Full Version?

If you want to set up the complete system with backend services:

```bash
# Install dependencies
npm install

# Set up the backend
cd backend
npm install

# Start the backend server
npm start

# In another terminal, start the frontend
npm run start
```
